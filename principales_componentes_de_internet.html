Principales componentes de Internet

Clientes y servidores 

El cliente es quien solicita información (por ejemplo, al usar un navegador), y el servidor es quien responde enviando los datos solicitados.

Proveedores de servicio (ISP)

Son las compañías que proporcionan el acceso a Internet a hogares y empresas, como Movistar o Vodafone.

Nodos y routers 

Los nodos son los puntos de conexión en la red, y los routers se encargan de dirigir el tráfico de datos por el camino más eficiente.

Direcciones IP y DNS 

La dirección IP identifica a cada dispositivo en la red. El DNS traduce nombres web como ‘YouTube’ a direcciones IP para que los navegadores los entiendan.

Protocolos 

Son las reglas que permiten la comunicación entre dispositivos, asegurando que todos hablen el mismo 'idioma' digital.
